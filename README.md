⚓ Ancla (Anchor)
Asistente de Navegación Sensorial y Cognitiva para el Espectro Autista.

Ancla es una aplicación móvil nativa para Android, desarrollada en Kotlin y Jetpack Compose. Este proyecto nace de la necesidad de crear una herramienta de apoyo digital de alta calidad, diseñada bajo principios de baja estimulación sensorial y previsibilidad.

🎯 Enfoque del Proyecto

El desarrollo de Ancla se centra en la perfección técnica y la utilidad real. Buscamos elevar las soluciones existentes mediante:

    Validación Clínica y Vivencial: Funcionalidades diseñadas a partir del feedback de una profesional de la salud mental que vive el espectro en primera persona.

    Arquitectura Nativa (Kotlin + Compose): Garantizando una respuesta inmediata y fluida, vital en momentos de sobrecarga sensorial.

    Diseño Centrado en la Calma: Una interfaz que rechaza el ruido visual y prioriza la paz mental del usuario.

✨ Funcionalidades en Desarrollo (Roadmap)

    🧩 Tablero de Tarea Única: Para reducir la parálisis por análisis y fomentar la autonomía.

    🔍 Decodificador de Contexto: Una herramienta lógica para ayudar a interpretar tonos y mensajes sociales ambiguos.

    🗣️ Scripts de Comunicación: Apoyo visual para momentos de mutismo selectivo o estrés elevado.

    🌙 Modo SOS (Refugio): Un entorno de baja estimulación con guías rítmicas para la autorregulación sensorial.

🛠️ Stack Tecnológico

    Lenguaje: Kotlin

    UI Framework: Jetpack Compose

    Diseño: Material Design 3 (Personalizado con paleta de colores muted).

    Arquitectura: MVVM (Model-View-ViewModel) para un código escalable y limpio.
